1 - Criar uma pasta para abrigar os arquivos, descompactar nela
2 - O pdfkit precisa do wkhtmltopdf instalado no sistema. Você pode baixar o binário de https://wkhtmltopdf.org/downloads.html.
3 - Rodar o código "pip install -r requirements.txt"
4 - Atualizar as credenciais no arquivo .env
5 - Atualizar a data no corpo do script dou.py
6 - Rodar o script primcipal dou.py com o código "python dou.py"
